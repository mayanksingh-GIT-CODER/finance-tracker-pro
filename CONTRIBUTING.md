# Contributing

Thanks for helping improve Finance Tracker Pro.

1. Open an issue describing the user problem or bug.
2. Create a focused branch from `main`.
3. Install development dependencies with `pip install -r requirements-dev.txt`.
4. Add or update tests for behavior changes.
5. Run `ruff check .` and `pytest`.
6. Submit a pull request describing the change, rationale, and verification.

Never commit real financial data, local databases, `.env` files, credentials, or generated exports.
